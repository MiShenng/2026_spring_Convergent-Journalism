溯源：飞天怎样从壁画母题生长为可传播的文化符号。

图表占位：历代飞天样式时序图（北凉至元）

回到传播发生之前，飞天为什么具备出圈条件？从北凉到元，姿态、飘带、色彩逐渐沉淀成一套稳定的样貌。这些经典符号让它天然适合被识别和再创作。

脆弱是真实的另一面。风化、起甲、酥碱吞掉了壁画上原本可看的部分。拖动滑块，直观感受保护与传播为何不可分。

飘带 反弹琵琶 敦煌色

飘带、反弹琵琶、供养场景和色彩系统被拆成可视元素。短视频里的飞天并非凭空生成，借走的从来不是固定图案，而是这套可被反复挪用的视觉语言。

破圈：平台、直播与二创怎样放大飞天的可见度。

图表占位：平台与内容形态矩阵（抖音、B站等 x 舞蹈、美妆等）

飞天从洞窟进入屏幕以后，平台是第一层放大器。在不同平台，飞天以舞蹈复刻、壁画科普、妆造教程等不同形态被疯狂传播。

直播和达人补上了传播的时间线。飞天的可见度不只靠十几秒的短视频，更靠这种持续几个小时的现场直播撑起传播主线。

国潮 敦煌舞 美学 AI复原 文化复兴

热门词条和评论流反映的不仅是热度高低，更是用户如何理解飞天。是国潮、美学、还是二创素材？不同语境赋予了飞天反复改写的意义。

图表占位：文博 IP 热度节奏对比（飞天 vs 故宫 vs 三星堆）

把飞天放进文博 IP 的节奏里。相较于故宫的常态平稳和三星堆的偶发尖峰，飞天正经历着越来越密的小波峰，慢慢走向常态被人看见的状态。

复兴：技术与消费怎样把飞天带入新的使用场景。

图表占位：数字重生流程图（8K采集 -> AI修复 -> 三维建模 -> AR互动）

传播热度的背后是技术链条。采集、修复、建模和互动展示，不仅改变了壁画的观看方式，更为创作者提供了源源不断的数字素材。

残损壁画被复原时，飞天从不可辨认的痕迹变成可观看、可讲述、可转译的图像。技术究竟给传播补回了多少内容，用眼睛感受。

从能被看清到能被买走。盲盒、汉服、美妆等文创品类构成了飞天 IP 经济的一面。年轻用户为文化符号付费意愿强烈。
